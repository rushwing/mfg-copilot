# Goal

Add operational triage, RCA support, and production hardening for multi-agent workflows.

# In Scope

- ops triage across logs, metrics, and build history
- RCA, FA, and escalation summaries
- eval, load, and observability hardening for operational workflows

# Out of Scope

- broad autonomous remediation
- ungoverned code-change automation

# Exit Criteria

- ops users can run grounded triage workflows
- RCA and escalation outputs are reviewable in the portal
- operational workflows have dedicated harness evals and load baselines
