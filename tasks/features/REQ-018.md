# User Story

As an approver or manager, I want every implementation PR to produce a standard review packet so I can review changes against a consistent structure instead of ad hoc summaries.

# Goal

Stabilize the repository-owned contract for PR review packets before workflow automation and merge gating depend on it.

# Deliverables

- PR review packet schema with required sections and identifiers
- artifact naming and storage convention for generated packets
- linked requirement and ADR reference format guidance
- generation-mode guidance for reviewer-facing packet outputs
- examples or fixture guidance for future packet producers

# In Scope

- packet schema shape
- required packet sections
- artifact naming and path convention
- linked requirement and ADR identifier format

# Out of Scope

- full workflow automation that generates packets for every PR
- UI rendering of review packets

# Acceptance Criteria

1. A repository-owned packet contract defines required sections such as summary, validation evidence, risks, checklist, and visuals.
2. The packet contract defines how linked REQ and ADR references should be represented so downstream generators do not invent competing formats.
3. Artifact naming or path guidance is clear enough that generated packets can be attached or stored consistently by future workflows.
4. The contract distinguishes machine-consumed structure from reviewer-facing prose so `draft_then_structure` generation can be applied intentionally.
5. Schema guidance or examples exist where future packet generators would otherwise drift.

# Validation Notes

- schema validation or fixture checks should run in CI where practical
- example packet fixtures should cover the minimum required fields before implementation automation begins

# Dependency Notes

- Depends on `REQ-003` because PR review packets should reuse the same shared-schema discipline as other cross-service and workflow artifacts.
- This story should land before packet generation logic or merge gating is implemented so those workflows target a stable contract.
